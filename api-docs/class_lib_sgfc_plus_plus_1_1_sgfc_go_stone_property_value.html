Detailed Description

The SgfcGoStonePropertyValue class provides an implementation of the ISgfcGoStonePropertyValue interface. See the interface header file for documentation.

LibSgfcPlusPlus::SgfcGoStonePropertyValue::SgfcGoStonePropertyValue ( std::shared_ptr< ISgfcGoStone >  goStone)

Initializes a newly constructed SgfcGoStonePropertyValue object. The object holds a Point value represented by goStone.

SgfcGoStonePropertyValue uses the SGF notation of the ISgfcGoPoint object referenced by goStone as the raw property value. This refers to the location of the stone on the board.

Exceptions
std::invalid_argumentIs thrown if goStone is nullptr.
