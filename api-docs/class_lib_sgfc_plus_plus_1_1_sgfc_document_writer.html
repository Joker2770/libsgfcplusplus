◆ GetArguments()

std::shared_ptr< ISgfcArguments > LibSgfcPlusPlus::SgfcDocumentWriter::GetArguments ( ) const
virtual

Returns an object with the collection of arguments that ISgfcDocumentWriter passes on to SGFC whenever it performs a write operation.

The collection of arguments is initially empty. Add arguments to the collection to change the way how SGFC writes SGF content. The collection retains its state between write operations so that repeated write operations use the same arguments.
