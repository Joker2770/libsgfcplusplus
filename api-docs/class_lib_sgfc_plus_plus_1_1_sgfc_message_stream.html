Detailed Description

The SgfcMessageStream class stores the content of the message stream that was generated by SGFC during its most recent load/parse or save operation.

SgfcMessageStream acquires the message stream content immediately upon construction, then clears the message stream in SGFC. This means that if two SgfcMessageStream objects are constructed in a row, the second object will see an empty message stream.
