Detailed Description

The ISgfcPointPropertyValue interface provides access to one of the values of a property of an SGF node. The value type is SgfcPropertyValueType::Point.

ISgfcPointPropertyValue is immutable, i.e. once the ISgfcPointPropertyValue object is constructed it cannot be changed.
