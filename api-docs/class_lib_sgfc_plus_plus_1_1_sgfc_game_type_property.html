◆ GetGameType()

SgfcGameType LibSgfcPlusPlus::SgfcGameTypeProperty::GetGameType ( ) const
virtual

Returns the property value interpreted as an SgfcGameType value.

Return values
SgfcGameTypeIf the property has a value defined in the SGF standard, returns the SgfcGameType value that corresponds to the property value. The value is guaranteed not to be SgfcGameType::Unknown.
SgfcConstants::DefaultGameTypeIf the property has no value. The value is guaranteed not to be SgfcGameType::Unknown. Note: The property having no value is not something that can naturally occur when SGF content is processed. It may occur in special circumstances, e.g. while a library clients programmatically sets up a game tree.
SgfcGameType::UnknownIf the property has a value that is not defined in the SGF standard. Invoke GetGameTypeAsNumber() to obtain the game type as Number value.

◆ GetGameTypeAsNumber()

SgfcNumber LibSgfcPlusPlus::SgfcGameTypeProperty::GetGameTypeAsNumber ( ) const
virtual

Returns the property value interpreted as an SgfcNumber value. This is useful if GetGameType() returns SgfcGameType::Unknown because the Number value is not defined in the SGF standard and cannot be mapped to a member of the enumeration SgfcGameType.

Return values
SgfcNumberThe Number value of the property. If the property has no value, returns the Number value that corresponds to SgfcConstants::DefaultGameType.
